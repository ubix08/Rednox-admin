# RedNox Visual Flow Editor\n\nThis repository contains the RedNox Visual Flow Editor built with React and TypeScript.\n\n## Project Structure\n\n- **public/**  \n  Contains static files like the HTML file and images.  \n- **src/**  \n  Contains the main application code.  \n  - **components/**  \n    Contains React components.  \n  - **hooks/**  \n    Contains custom React hooks.  \n  - **pages/**  \n    Contains different pages of the application.  \n  - **styles/**  \n    Contains global styles and CSS modules.  \n  - **utils/**  \n    Contains utility functions and constants.  \n- **.env**  \n  Contains environment variables.  \n- **.gitignore**  \n  Specifies files to ignore in Git.  \n- **package.json**  \n  Contains project metadata and dependencies.  \n- **tsconfig.json**  \n  TypeScript configuration file.  \n- **webpack.config.js**  \n  Webpack configuration file for building the project.  \n\n## Getting Started\n\n1. Clone the repository.  \n2. Change directory into the project folder.  \n3. Install dependencies by running `npm install`.  \n4. Start the development server by running `npm start`.\n
